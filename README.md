<hr \>

## Setup
## Clone the repository

```
git clone https://github.com/rxsemxrriee/TEE-AOF && cd TEE-AOF
```
## Backend setup
```
cd backend
go run main.go
```
## Frontend setup
```
cd frontend
npm install
npm run dev
```

<hr \>

## Techstacks:
```
Frontend: React.tsx
Backend: Golang
Database: MySQL
Realtime: Websockets
```
